Connect MarsClaw to messaging platforms. Add channels via CLI or configure here.

How to add channels
1
Run marsclaw channels add in your terminal
2
Select a provider (Telegram, Discord, Slack, WhatsApp)
3
Paste your bot token — MarsClaw saves it securely to ~/.marsclaw/channels.json
4
Run marsclaw telegram or marsclaw serve to go live
